Memory
!memories
!memories
View Luna's 4 layers of memory about you (core, long-term, short-term, working).
!remember
!remember <fact>
Store a fact in long-term memory. Example: !remember I prefer dark mode.
!always_remember
!always_remember <fact>
Store an essential fact in core memory (e.g. name). Example: !always_remember My name is Alex.
!forget
!forget
Clear long-term and short-term memories. Core memory is kept.
!forget_all
!forget_all
Clear all memories including core.
